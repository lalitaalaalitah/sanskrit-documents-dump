Sanskrit Romanized Keyboard Layout Help

 

1) This keyboard driver was made using Microsoft Keyboard Layout Creator. It is designed for those who by some reasons can not use Devanāgarī Inscript Keyboard Layout. The main principle of typing in this Romanized layout remains the same as in Inscript Layout but all the keys were reassigned to phonetically similiar Roman keys (were it was possible). For example, क (the letter "ka") must be typed as "k".

2) This keyboard layout is designed specially for Sanskrit Language and supports some Vedic characters that can be viewed only in Sanskrit2003 font. Other essential feature of this layout is the use of "Right Alt" key (AltGr).

Here are graphical and text instructions for using Sanskrit Romanized Keyboard Layout.

How to Install Sanskrit Romanized Keyboard Layout?

1) First make sure that support of Devanāgarī Script is enabled in Your Sustem (this can be done only in Windows 2000, Windows XP and Windows 2003). To do this go to Control Panel > Language Settings > Languages > and "Enable support of complex and left to right scripts"

2) Click on Setup.exe.

3) Then go to Control Panel > Language Settings > Languages > Languages and Text Input Services > Add > and select as language Sanskrit and Sanskrit Romanized as IME.

 

How to Type Devanāgarī?

Here are some examples:

क (k) + ट (q) = कट (kaṭa)

 

क (k) + ् (/) = क् (k)

क (k) + ा (A) = का (kā)

क (k) + ि (i) = कि (ki)

 

क (k) + ॑ (!) = क॑ (kà)

 

क (k) + ् (/) + ष (S) = क्ष (kṣa)

र (r) + ् (/) + क (k) = र्क (rka)

द (d) + ् (/) + द (d) = द्द (dda)

 

ल (l) + ् (/) + (? - Zero Width Joiner) + ँ (Alt+m) + ल (l) = ल्‍ँल (l̐la)
