Transliteration code

The code used here for transliteration of Sanskrit words is the ITRANS 4.0 used in internet with a few modifications.

The Sanskrit vowels are, in order, as below:-

a, aa, i, ii, u, uu, r., r.., l., l.., e, ai, o, au.

Consonants:--

kavarga--- k, kh, g, gh, .n

chavarga—ch, chh, j, jh, nj

Tavarga—T, TH, D, DH, N

tavarga--- t, th, d, dh, n

pavarga—p, ph, b, bh, m

y, r, l, v, s’, sh, s, h

visarga--H
