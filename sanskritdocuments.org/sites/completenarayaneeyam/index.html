It has been tested and found to work with Internet Explorer Version 8.0, 9.0 and 10.0, Google Chrome Version 39, Mozilla Firefox Version 34, and Safari Version 5. However, with Internet Explorer 8.0, the auto-forward feature to move automatically to the next shlokam does not work. The user must click the "Next" button to move forward.

The contents are the same as the original version.

However, the appearance of the screen is changed to some extent.

This version adheres to HTML 5 and uses some of its new features for serving the audio content.
