Sanskrit Documents

File Formats 
.itx         ITX:- ITRANS input/encoded etext document, ready for processing
                  to print in Devanagari or many different scripts.  These files can 
                  be directly viewed in Devanagari or Roman international transliteration
                  using Itranslator for PC with version 98, 99, 2003  or
                  converted to postscript format using Itrans.
.gif         GIF:- GIF image can be viewed by any web browser.
.html        XDVNG, ROMAN, UNIC 
                  Either standard hyper text or display file using various fonts (mostly XDVNG Devanagari),
                  or  Roman+diacritics, or Unicode UTF that can be seen in different
                  Indian scripts using http://girgit.chitthajagat.in/
.BRH         BRH:- Baraha input/encoded etext document, ready for processing
                  using Baraha Indian language processor.
_sans98.html SANS98:- HTML Devanagari file with Vedic accents or
                  svaras in Sanskrit98 fonts as input to Itranslator98.
                  The Sanskrit98 are available at Itranslator98 site.
_sans98.htm  TRNS98:- HTML roman+diacritics file with Vedic accents or
                  svaras in Translit98 fonts as input to Itranslator98.
                  The Translit98 are available at Itranslator site.
_sans99.html SANS99:- HTML Devanagari file with Vedic accents or
                  svaras in Sanskrit99 fonts as input to Itranslator99.
                  The Sanskrit99 are available at Itranslator site.
_sans99.htm  TRNS99:- HTML roman+diacritics file with Vedic accents or
                  svaras in URW Palladio IT fonts as input to Itranslator99.
                  The URW Palladio IT fonts are available at Itranslator site.
.ps          PS:- Postscript file for postscript printer or postscript viewer.
                  Requires combination of Ghostview and Ghostscript.
.pdf         PDF:- Portable Document Format file
                  to view using Adobe Acrobat Reader, a viewer available here.
.pdf         SPDF:- Searchable Portable Document Format file
                  to view using Adobe Acrobat Reader, a viewer available here. 
                  The files support Sanskrit98/99 fonts and allow easy search. 
                  (Contributed by Ulrich Stiehl.)
.tex         TEX:- LaTeX input file.
.txt/.asc    Unformatted ITRANS input/encoded etext file.
.txt_sans98  TXT98:- Text file with Vedic accents or svaras as 
                  input to Itranslator98/99.
.s           Includes Sanskrit transliterated document
.doc         Microsoft Word document DOC.
.rtf         Microsoft Word document Rich Text Format.
.Z           Unix compressed file.
.gz          Unix GZIP compressed file.
.tar         Archive file (UNIX tar) containing many embedded files.
.zip/.ZIP    PC ZIPped file
