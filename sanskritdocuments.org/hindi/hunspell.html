देवनागरी में लिखी जाने वाली भाषाओं के वर्तनी जांचक शब्दकोष (Hunspell format)

Spell Checking Dictionaries

Hunspell format affix files and dictionaries, based on wordlists available on the internet are being provided here in the hope that they will be useful for the users of the language to spellcheck their documents.

We will be adding more dictionaries to this page over the next few weeks. See README.txt in the zip files for list of sources used.

Use with Notepad++

The .dic and .aff files can be used in notepad++ with DSpellCheck plugin. Read the DSpellCheck manual for detailed instructions.

Please add \u0964 \u0965 \u0966 \u0967 \u0968 \u0969 \u096A \u096B \u096C \u096D \u096E \u096F to the delimiters under the Advanced tab in DSpellCheck Settings, so that hindi numerals (० १ २ ३ ४ ५ ६ ७ ८ ९) and punctuation (। and ॥) are also recognized.

Use with other programs

It should also be possible to use the zip files as extension in openoffice/libreoffice as well as other programs that support hunspell format dictionary files.

The .dic and .aff files can be copied to the appropriate dictionary folders replacing existing files (make sure to rename the old files as backup in case you want to go back to the old version).
