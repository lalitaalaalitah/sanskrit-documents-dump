  • Ajit Krishnan has developed "amara kosha" website at http://amara.aupasana.com
    This is built almost entirely on the data made available by the University of Hyderabad amara kosha project. It is surfaced differently, giving more importance to the amara kosha shlokas themselves. There's a link to the github repo at the bottom of the page.
    Functionality:
    • View any varga in it's entirety, along with the meanings
    • Search for any word in itrans / devanagari with wildcards
    • Vocabulary builder via quizzes
    • Telugu translations of svarga varga available if telugu output is selected
    • The site can be viewed in devanagari, telugu, kannada or malayalam scripts
    • Implementing audio playing utility (available up to kAla varga March 2019)
    • Includes a word list based on starting letter, a list of all word duplicates, and basic word / line count statistics

    Data set at University of Hyderabad devanagari csv file at https://raw.githubusercontent.com/aupasana/amara-quiz/master/database/amara_pada.csv. There is a PhD thesis The Knowledge Structure in Amarakosha by Sivaja S. Nair, 2011.
