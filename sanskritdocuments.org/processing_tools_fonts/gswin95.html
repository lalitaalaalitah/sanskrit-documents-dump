Ghostscript & GSview - Postscript Viewer

    This note is regarding installation of postscript viewer called GSView on a PC with Windows. This allows displaying devanaagarii / Sanskrit / Marathi / Hindi files generated using ITRANS.

    GSview is a graphical interface for Ghostscript under MS-Windows. Ghostscript is an interpreter for the PostScript page description language used by laser printers for documents following the Adobe PostScript Document Structuring Conventions, GSview allows selected pages to be viewed or printed even on non-postscript printers using the mswinpr2 printer driver in GSview. Aladdin Ghostscript 4.03 is required for using GSview.

    The software is available free. Please check http://www.cs.wisc.edu/~ghost/index.html for the latest information about the package. It is also available for Unix, VMS, DOS, OS/2 and mac environments.

    Please refer to http://www.cs.wisc.edu/~ghost/aladdin/ for details.

    Obtaining latest Aladdin Ghostscript

    The documentation for Ghostscript is in the gs503ini.zip file. Look for README, new-user.txt, install.txt and use.txt.

    For GSView:

    Obtaining GSview

    GSview contains an installation program which also installs Ghostscript . See the README.TXT file for more details.

    Installing GSview under Windows is real easy, everything installs very easily in about 5 minutes from the start of setup.

    • Just make a temporary directory on your hard drive for the installation (call it GSTEMP)
    • Get all the 4.zip files required for installing Ghostscript & GSview and copy to GSTEMP directory.
    • Using winzip or pkunzip, unpack just gsview21.zip, and then run the setup program there.
    • The GSview setup program will automatically unzip all the other zips, and install and configure the entire Ghostscript/Ghostview package in a gstools location.
    • If you go with the default options when asked fro choices, GSview should install and work fine for you.
    • Delete the files in the temporary directory GSTEMP. Delete GSTEMP directory also.
